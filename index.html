Projet

Site Realiseo

Je travail à la réalisation d'un site web pour Réaliseo, une agence spécialisée dans le conseil, la formation et l'évaluation en matière de Responsabilité Sociétale des Entreprises (RSE) et d'Environnement, Social et Gouvernance (ESG).

À partir de la charte graphique et en étroite collaboration avec M.Billet, proprétaire de l'entreprise, une première version fixe du site est presque achevée.

Je poursuit actuellement le developpement avec la construction d'un back-end pour intégrer de nouvelles fonctionalitées : formulaire de contacte, page de blog et paiment en ligne. Je prévois égallement d'utiliser un framework ou une librairie Javascript pour le front-end.

  • HTML
  • CSS
  • Node JS
  • Express
  • EJS

Site vitrine / e-commerce d'artiste

Il s'agit de mon projet de fin de formation, une site web qui permets à un artiste d'exposer et de vendre son travail. Il s'agit d'un prototype, l'application est appelée à être améliorée.

Les visiteurs ont l'a possibilité de se créer un compte client pour accèder aux fonctionalitées de boutique en ligne : séléctioner des produits en vente dans un panier et les achéters, passer une commande personalisée directement aux près de l'artiste.

Le propriétaire du site à accès à un éspace administrateur qui luis permets de gérer les produits en ventes, les achâts et les commandes des clients.

  • HTML
  • CSS
  • Node JS
  • SQL
  • Express
  • EJS
